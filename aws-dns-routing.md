# aws-dns-routing

Types of Route53 routing:
* Simple 
* Weighted
* Latency-based
* Failover
* Geolocation
* Geoproximity 
* Multi-value answer

## Simple Routing Policy

One entry with multiple IP addresses. Route53 randomly selects the IP address to return for each query.
